# custom-merch
Day 1 of 10 days of Frontend challenge by Codevixens
The goal today was mastering html forms and various input types while ensuring proper form structure and improving user experience
Elements included: Text input, password input, radio buttons, checkboxes, a dropdown menu, and a submit button.
The task was to create an HTML file and structure a basic form, add the required input fields based on the provided elements, style the form using basic CSS for a clean layout, and test the form functionality by submitting it.
